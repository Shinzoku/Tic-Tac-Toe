# Tic-Tac-Toe Game

## Introduction

This project is a simple Tic-Tac-Toe game that allows two players to play against each other on the same device. The game is implemented using HTML, CSS, and JavaScript. The goal is to get three of your marks in a row (horizontally, vertically, or diagonally) before your opponent does.

## Project Structure

The project is organized into three main files:
- `index.html`: Contains the HTML structure of the game.
- `style.css`: Contains the CSS styles for the game layout and design.
- `script.js`: Contains the JavaScript logic for the game functionality.

## Cloning the Project

```bash
    git clone git@github.com:Shinzoku/Tic-Tac-Toe.git
```

+ HTTPS :
```bash
    git clone https://github.com/Shinzoku/Tic-Tac-Toe.git
```

## Features

+ Two-Player Mode: Players can take turns clicking on the cells to place their marks (X or O).
+ Winning Detection: The game detects when a player has won and displays an alert.
+ Draw Detection: The game detects when all cells are filled without a winner and displays an alert for a draw.
+ Reset Functionality: A button to reset the game and start over.
